- in the header they have an image of their logo which is centered and linked to their home page
- for their hamburger bar, they use the svg tag and actually made each line themselves in code rather than using an image (which I would've been tempted to do because I'm hardwired to see elements like that as images)
- in main content they have the main image of the product and then a slider below it, with is actually made up of an ordered list and list elements
